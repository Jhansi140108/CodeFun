# CALBOT
CALBOT is a simple C-based interactive agent that helps users solve basic mathematical problems. Users can select from a list of available options, and CALBOT provides the answer based on the chosen operation.

## Features
* User-friendly menu-driven interface
* Performs basic mathematical calculations
* Quick and simple interaction through an agent named **CALBOT**
* Suitable for beginners learning C programming

## Available Operations
* Addition
* Subtraction
* Multiplication
* Division
* areas of basic shapes
* perimeters of basic shapes
* Other basic mathematical operations (if implemented)

## Requirements
* GCC Compiler or any standard C compiler

## How to Compile
```bash
gcc calbot.c -o calbot
```

## How to Run
### Linux/macOS
```bash
./calbot
```
### Windows
```bash
calbot.exe
```
## Example
```
Welcome to CALBOT!

Choose an operation:
1. Addition
2. Subtraction
3. Multiplication
4. Division

Enter your choice: 1
Enter two numbers: 10 20

CALBOT: The answer is 30
```

## Project Objective
The goal of CALBOT is to provide a simple mathematical assistant that demonstrates the use of C programming concepts such as functions, conditional statements, loops, and user interaction.
